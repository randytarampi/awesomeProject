About This "App"

If you're looking at the source code for this page on our GitHub Repo then you might have noticed that this page (more like template for this page) exists in a different templates directory - the one inside the "about" app, not the one in the project's root directory. This is because of portability reasons. Just in case you want to recycle a particular Django app, i.e. port it to another app, but want to keep the view (template) the same, you can. You can even control the granularity like we did, making this page rely on our project's "base.html".

Cool stuff eh?
